oh-my-zsh-addons
================

Installation
------------
```
git clone git@github.com/fhwrdh/oh-my-zsh-addons
ln -s ~/oh-my-zsh-addons/thems/fhwrdh.zsh-theme ~/.oh-my-zsh/themes/fhwrdh.zsh-theme
ln -s ~/oh-my-zsh-addons/custom/alias.zsh ~/.oh-my-zsh/custom/alias.zsh
ln -s ~/oh-my-zsh-addons/custom/autojump.zsh ~/.oh-my-zsh/custom/autojump.zsh
ln -s ~/oh-my-zsh-addons/custom/functions.zsh ~/.oh-my-zsh/custom/function.zsh
ln -s ~/oh-my-zsh-addons/custom/history.zsh ~/.oh-my-zsh/custom/history.zsh
